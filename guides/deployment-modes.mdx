## Overview

FlowWarden supports three deployment strategies for multi-instance environments:

- **SINGLE_LEADER** — One active instance, others on standby (default)
- **ALL_INSTANCES** — Every instance receives all events
- **PARTITIONED** — Horizontal scaling with document key-based distribution
