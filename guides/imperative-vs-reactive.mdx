## Overview

FlowWarden supports two execution modes configured via `flowwarden.default-mode`:

- **IMPERATIVE** — uses `MongoTemplate`, blocking handlers
- **REACTIVE** — uses `ReactiveMongoTemplate`, returns `Mono<Void>` or `Flux`
