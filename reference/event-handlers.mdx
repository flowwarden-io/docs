## Overview

Typed event handler annotations let you react to specific Change Stream operation types.

| Annotation | Triggers on |
|-----------|-------------|
| `@OnInsert` | Document inserts |
| `@OnUpdate` | Document updates |
| `@OnDelete` | Document deletes |
| `@OnChange` | All operation types |
