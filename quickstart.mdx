## 1. Add the dependency

```xml
<dependency>
  <groupId>io.flowwarden</groupId>
  <artifactId>flowwarden-stream-core</artifactId>
  <version>0.1.0-SNAPSHOT</version>
</dependency>
```

## 2. Configure your mode

```yaml
flowwarden:
  default-mode: IMPERATIVE  # or REACTIVE
```

## 3. Enable FlowWarden

```java
@SpringBootApplication
@EnableFlowWarden
public class MyApplication { }
```

## 4. Create a handler

```java
@ChangeStream(collection = "orders")
public class OrderStreamHandler {

    @OnInsert
    public void onNewOrder(ChangeStreamContext<Order> ctx) {
        Order order = ctx.getBody();
        // process new order
    }
}
```
